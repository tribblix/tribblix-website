--- title: ISO Downloads ---

> Download

The ISO images are available here:

For 64-bit x86/x64 systems:

Milestone m36.1, standard image

  • Tribblix 0m36.1 (sha1 hash: 383e3e5c828c6b0c9cee23bd6b70a16100489361) - Release Notes
  • OmniTribblix 0m36.1, with LX zones (sha1 hash: 4f2014235cdaf3a9daa936eea3027832c7654526) - Release Notes

Milestone m36.1, minimal/server image

  • Tribblix 0m36.1 (sha1 hash: d9ee125b43f4044d86df0d450542d14cd9f4df59) - Release Notes
  • OmniTribblix 0m36.1, with LX zones (sha1 hash: c0b49c919acee1c0bf457807c60be34216d8d8b8) - Release Notes

For SPARC systems

Milestone m32, smaller image for CDROM use

  • Tribblix 0m32 SPARC (sha1 hash: 6d0b2cc6e008a9da24335f1232b3d6772eb865a5) - Release Notes

Milestone m31, standard image

  • Tribblix 0m31 SPARC (sha1 hash: 99c794de0c2c1caf02dceb9e24692085608c455b) - Release Notes

The current ISO for SPARC is far more stable and supportable than early ISO images, but do check the Release Notes before proceeding.

For Cloud

You don't need to download the ISO, you can run Tribblix in the cloud.

Some older images are also available.
