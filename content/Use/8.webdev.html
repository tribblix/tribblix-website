> Web Development Application Notes

A general selection of utilities associated with web development is available in the webdev overlay.

zap install-overlay webdev

The imaging overlay may also be relevant, as it includes some image optimisation utilities such as pngquant, optipng, and jpegoptim.

Depending on your stack, you may wish to look at the java11, java17, java21, or node-v20 overlays.

KISS

If you want to use the KISS web development framework then, in addition to installing java, you'll need to install libsqlitejdbc:

zap install TRIBlibsqlitejdbc

Reposilite

If you want to use the Reposilite repository manager then, in addition to installing java, you'll need to install libsqlitejdbc:

zap install TRIBlibsqlitejdbc
