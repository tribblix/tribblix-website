> Building Tribblix

One of the key freedoms of free software is the freedom to study how the program works, and change it so it does your computing as you wish (freedom 1). While there's always a requirement for a non-trivial level of skill, the intention of this chapter is to explain how you can exercise this freedom in the context of Tribblix.

The first step in this is to understand the list of source code repositories that are used to construct Tribblix.

The second step is to build illumos-gate from source.

The third step is to create packages for third-party software.

The fourth step is to create release packages for a new release, and for upgrades from earlier releases.

The fifth step is to manage the overlays.

The sixth step is to create ISO images.
