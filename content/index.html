--- title: the retro illumos distribution ---

>_

Tribblix is an open source operating system created by Peter Tribble. Based on illumos, it blends a retro style with modern components.

NEWS: February 27th 2025: Milestone 36.1 available. This is a refresh of the m36 functional release to fix possible network driver issues. (updates). Vanilla Tribblix and LX (aka "omnitribblix") variants for x86 are now available. (download - X86 Release Notes.)

NEWS: October 15th 2024: Milestone 31 ISO available for SPARC. (download - SPARC Release Notes.)

Important: 32-bit support.

While the SPARC version still suffers from lack of testing and has spotty hardware support, the x86 version is pretty solid. You can download the image, install to a hard drive, and use it.
