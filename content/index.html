--- title: the retro illumos distribution ---

>_

Tribblix is an open source operating system created by Peter Tribble. Based on illumos, it blends a retro style with modern components.

NEWS: July 13th 2025: Milestone 37 available. (updates). Vanilla Tribblix and LX (aka "omnitribblix") variants for x86 are now available. (download - X86 Release Notes.)

NEWS: July 29th 2025. SPARC upgrades from m32 to m33 now available, for now fresh installs should use the m32 ISO and run an upgrade. (SPARC Release Notes.)

NEWS: May 15th 2025. CDROM sized ISO for m32 SPARC now available. (download - SPARC Release Notes.)

Important: 32-bit hardware support now completely removed.

While the SPARC version still suffers from lack of testing and has spotty hardware support, the x86 version is pretty solid. You can download the image, install to a hard drive, and use it.
